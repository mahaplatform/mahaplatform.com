Start new conversation

You can begin as many chat conversations as you like. Conversations can be started from the sidebar or the chat window

From the Ribbon

  1. Click the chat icon in the ribbon
  2. Click the plus icon in the upper right hand corner of the sidebar
  3. Select members for this conversation
  4. Click Start

From the Menu

  1. Click the menu icon in the ribbon
  2. Click Chat
  3. Click the plus icon in the upper right hand corner of the conversations panel
  4. Select members for this conversation
  5. Click Start
